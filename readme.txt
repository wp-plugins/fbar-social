== Description ==

FBar Social plugin is a social sticky header bar for wordpress that magically appears at a specific pixel.

= More information =
Please visit the plugin website at http://www.geckoandfly.com/ for more information.

== Installation ==

The easiest way to install FBar Social is via your WordPress Dashboard. Go to the "Plugins" page and search for "FBar" in the WordPress Plugin Directory. Then click "Install Now" and the following steps will be done for you automatically. You'll just have to activate FBar Social.

Manual installation works just as for other WordPress plugins:

1. Download and extract the ZIP file and move the folder "fbar" into the "wp-content/plugins/" directory of your WordPress installation.
1. Activate the plugin "fbar" on the "Plugins" page of your WordPress Dashboard.
1. Make changes to the settings and let the magic begins.


= Requirements? =

In short: WordPress 3.5 or higher, while the latest version of WordPress is always recommended.

== Usage ==

No widget and no codes to add to your theme, just activate the plugin and make the necessary changes to the settings.
